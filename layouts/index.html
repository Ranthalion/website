reef-pi

Welcome to the official website of reef-pi - An opensource reef tank controller based on Raspberry Pi. reef-pi is a modular, affordable and extendable DIY controller that can automate day to day reef keeping chores such as auto top off, temperature control and dawn to dusk light cycle.
If you are building a new controller, start with general guides which covers the main controller installation, then use the module specific guides to configure individual modules. You can follow reef-pi's ongoing development in the master thread at reef2reef. Contact at @{{ .Site.Params.Social.Twitter }} or by email. Website specific bugs & improvements can be reported here. If you are interested in reef-pi software development, use the github repository and slack channel to get started.
